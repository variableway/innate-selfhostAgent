# 阅读增强脚本 | Reading Enhancement Scripts

本目录包含用于提升 README 阅读体验的 JavaScript 脚本。

This directory contains JavaScript scripts to enhance README reading experience.

## 脚本列表 | Scripts

### 1. back-to-top.js
返回顶部按钮，当页面滚动超过 300px 时显示。

Back to top button that appears when scrolling past 300px.

### 2. smooth-scroll.js
为页面内锚点链接添加平滑滚动效果。

Adds smooth scrolling for anchor links.

### 3. reading-progress.js
页面顶部显示阅读进度条。

Shows reading progress bar at the top of the page.

## 使用方法 | Usage

在 HTML 文件中引入这些脚本：

Include these scripts in your HTML file:

```html
<script src="scripts/back-to-top.js"></script>
<script src="scripts/smooth-scroll.js"></script>
<script src="scripts/reading-progress.js"></script>
```

或者在文档站点配置中引入。

Or include them in your documentation site configuration.

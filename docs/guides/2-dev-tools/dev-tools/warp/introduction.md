# Warp Terminal

Warp is a blazingly fast, Rust-based terminal reimagined from the ground up.

## English Introduction

### Installation

**Windows**:
As of now, Warp is primarily available on macOS. Windows support is in development. Please check the official Warp website for the latest updates on Windows availability.

**macOS**:
You can install Warp via `brew` or by downloading it from the official website.
```bash
brew install --cask warp
```

### Key Features
*   **Blocks**: Commands and their output are grouped into blocks, making it easy to navigate and copy.
*   **AI Command Search**: Natural language to command translation.
*   **Collaborative**: Share your terminal sessions with your team.

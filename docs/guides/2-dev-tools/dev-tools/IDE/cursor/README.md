# Cursor IDE

Cursor is an AI-first code editor powered by GPT-4 that enables developers to build software faster with AI assistance.

## Features
- Inline AI chat and editing
- Context-aware code suggestions
- Multi-file awareness
- Real-time collaboration with AI

## Installation
1. Visit https://cursor.sh/
2. Download the appropriate version for your OS
3. Install and launch the application

## Plugin Installation
Cursor doesn't require additional plugins as it has AI capabilities built-in. However, you can install language-specific extensions:
1. Open Settings (Ctrl/Cmd + ,)
2. Navigate to Extensions
3. Search and install desired extensions

## AI Model Configuration
To configure AI models in Cursor:
1. Go to File > Preferences > Settings
2. Click on "AI Settings"
3. Configure your API key for enhanced capabilities
4. Adjust settings for context window and response preferences

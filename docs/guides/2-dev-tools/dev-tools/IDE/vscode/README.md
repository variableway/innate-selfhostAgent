# Visual Studio Code (VSCode)

Visual Studio Code is a popular, lightweight but powerful source code editor that supports AI integration through various extensions.

## Features
- Syntax highlighting for numerous languages
- Integrated Git control
- Customizable workspace
- Rich ecosystem of extensions

## Installation
1. Visit https://code.visualstudio.com/
2. Download and install VSCode for your operating system
3. Launch the application

## Plugin Installation
VSCode requires plugins for AI capabilities:
1. Open Extensions view by clicking the Extensions icon in the Activity Bar or pressing Ctrl+Shift+X
2. Search for AI-related extensions such as:
   - GitHub Copilot
   - Tabnine
   - Amazon CodeWhisperer
   - Codeium
3. Click Install on your chosen extension
4. Restart VSCode if prompted

## AI Model Configuration
After installing an AI extension:
1. Open VSCode Settings (Ctrl+, or Cmd+, on Mac)
2. Search for the installed AI extension settings
3. Configure your API key or sign in to the service
4. Customize preferences like auto-completion behavior

# Zed IDE

Zed is a high-performance, AI-powered code editor designed for professional developers with a focus on speed and efficiency.

## Features
- GPU-accelerated UI
- AI-assisted code completion
- Built-in terminal and debugger
- Collaborative coding features

## Installation
1. Go to https://zed.dev/
2. Download the installer for your platform
3. Install and launch Zed
4. Sign in or create an account to sync settings

## Plugin Installation
Zed has AI capabilities built-in, but you can enhance functionality with extensions:
1. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) to open command palette
2. Type "Install Extension"
3. Browse or search for extensions
4. Click Install to add them

## AI Model Configuration
Zed integrates with various AI providers:
1. Go to Settings > Preferences
2. Navigate to the AI section
3. Configure your preferred AI provider and API key
4. Adjust settings for completions and chat behavior

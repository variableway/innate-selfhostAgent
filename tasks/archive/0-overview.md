### Key Architectural Difference Between Two App Versions

| Aspect | `apps/desktop/` (older) | `playground/apps/desktop/` (newer) |
|--------|------------------------|-------------------------------------|
| Data model | `Tutorial` + `Series` | `Skill` + `Course` |
| Data source | Hardcoded `mockTutorials`/`mockSeries` in store | Dynamic scan of MDX files via `tutorial-scanner` |
| Storage | In-memory zustand | Persisted via `tauriStorage` |
| Content files | Not used | MDX files in `public/tutorials/` with frontmatter |
| Terminal | Mock output | Real PTY via Tauri (`invoke("pty_write")`) |
| Framework | Plain React | Next.js |

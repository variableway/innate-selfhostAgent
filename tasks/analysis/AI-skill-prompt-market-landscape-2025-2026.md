# AI Skill Market & Prompt Market Landscape Analysis (2025-2026)

> Research date: 2026-04-16
> Purpose: Comprehensive intelligence on platforms where AI skills, agents, workflows, and prompts can be published, shared, and monetized.

---

## Executive Summary

The AI skill/agent/prompt marketplace ecosystem has exploded in 2025-2026. The global AI prompt marketplace alone is projected at $2.51 billion in 2026 (up from $1.94B in 2025), with a CAGR of 29.5%. China's enterprise AI agent market has surpassed 23.2 billion RMB. Every major AI company (OpenAI, ByteDance, Anthropic, Google, Baidu, Alibaba) has launched or is launching a skill/agent marketplace.

**Key trend**: The market is shifting from "prompt marketplaces" (selling text strings) to "skill/agent marketplaces" (packaging complete workflows with tools, knowledge, and reasoning). Platforms that offer the lowest barrier to both creation and consumption are winning.

---

## Part 1: AI Skill/Agent Marketplaces

### 1. OpenAI GPT Store / ChatGPT GPTs

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | 700M+ weekly active ChatGPT users; millions of custom GPTs created; custom GPT market estimated at $3.7B opportunity |
| **Barrier to publish** | LOW - Anyone with a ChatGPT Plus/Pro subscription can create a GPT via natural language conversation. No coding required. |
| **Direct use by end users** | HIGH - Users discover GPTs in-store and start chatting immediately. Zero setup. |
| **Monetization** | GPT Revenue Program (pooled model based on engagement); ChatGPT Plus/Pro/Enterprise subscriptions required for creation |
| **Traffic/User base** | Massive - 700M weekly active users, 2.5B daily prompts, 600K+ Enterprise paying users |
| **Chinese equivalent** | Coze/扣子 is the closest analog |

**Strengths**: Largest user base, frictionless "discover and use" experience, natural language creation
**Weaknesses**: Revenue model for creators is opaque (pooled, not per-sale); limited customization depth; no standalone deployment
**Verdict**: Best "see and use" experience in the global market. Essential distribution channel.

### 2. Coze / 扣子 (by ByteDance)

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | Major platform in China; expanding globally via coze.com |
| **Barrier to publish** | LOW - No-code bot builder; natural language skill creation ("one sentence to generate a skill") |
| **Direct use by end users** | HIGH - Bots can be used directly in Doubao (豆包), WeChat, Feishu (飞书), and web |
| **Monetization** | Skill Store / Workflow Store with monetization; emerging creator economy |
| **Traffic/User base** | Very large in China via ByteDance ecosystem (Douyin, Feishu, Doubao integration) |
| **Key differentiator** | World's first "AI Skill Store" - launched with Coze 2.0; Workflow Store for sharing/remixing workflows |

**Coze 2.0 Key Features**:
- **Skills Marketplace**: Package professional expertise into reusable skill modules
- **Workflow Store**: Public sharing and remixing of AI workflows
- **Agent Skills**: Deep encapsulation of scenario-based best practices (e.g., marketing copywriting)
- Multi-platform distribution: bots deploy to Doubao, WeChat, Feishu, web

**Strengths**: Deepest "skill packaging" concept; workflow remixing; massive Chinese distribution; free tier available
**Weaknesses**: International version still growing; brand recognition outside China limited
**Verdict**: Most innovative skill marketplace concept globally. The "Skill Store" model is the direction the entire industry is moving.

**Weaknesses**: End-user friction - requires Dify deployment; not a consumer marketplace

### 4. n8n

| Dimension | Assessment |

| **Scale** | 9,167+ community workflow templates (official); 7,000+ on third-party sites |
| **Barrier to publish** | MEDIUM-LOW - Visual node-based editor; template sharing via community |

**Verdict**: Best workflow automation marketplace for technical users. Not a consumer "see and use" platform.

### 8. Anthropic Claude Skills / Extensions

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | Skills launched October 2025; ecosystem now has "thousands" of community skills |
| **Barrier to publish** | LOW-MEDIUM - Skills are defined in markdown/CLAUDE.md files; plugin marketplace system emerging |
| **Direct use by end users** | LOW for consumers; HIGH for Claude Code developers - install via `claude plugin marketplace add` |
| **Monetization** | No official monetization yet; community-driven sharing via GitHub repos |
| **Traffic/User base** | Claude Code developer community; growing rapidly |
| **Chinese presence** | Limited |

**Key developments**:
- Official Anthropic guide on [building skills for Claude](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf)
- Plugin system emerging that bundles slash commands + skills + MCP connectors
- Community directories (e.g., claudemarketplaces.com) curating skills
- AutoClaw (澳龙) - local client with 50+ built-in skills (from 智谱/GLM ecosystem)

**Strengths**: Official Anthropic support; markdown-based (very low barrier); growing ecosystem
**Weaknesses**: Developer-focused (Claude Code); no consumer marketplace; no monetization infrastructure
**Verdict**: Best skill definition format for developers. Not yet a consumer-facing marketplace.

## Part 2: Prompt Marketplaces

### 1. PromptBase

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | 100,000+ prompts; ~$1.9M revenue; London-based, founded 2022 |
| **Barrier to publish** | LOW - Register, upload prompts, set price |
| **Direct use by end users** | MEDIUM - Purchase prompt, copy-paste into AI tool |
| **Monetization** | 80/20 revenue split (creator keeps 80%); 10% on custom jobs |
| **Traffic/User base** | Largest dedicated prompt marketplace |
| **Chinese equivalent** | None dominant; AiShort serves Chinese community |

**Verdict**: Pioneer and market leader in prompt marketplaces. But "prompt selling" is increasingly commoditized as models get better at zero-shot tasks.

### 2. FlowGPT

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | $1.9M revenue (2025); "millions of monthly active users"; $10M pre-Series A funding |
| **Barrier to publish** | LOW - Free community sharing |
| **Direct use by end users** | HIGH - Chat with prompts directly on platform; no setup |
| **Monetization** | Freemium; community-driven; evolving toward paid features |
| **Traffic/User base** | Largest open-source AI prompt/app creator community |

**Verdict**: Best "discover and immediately use" prompt platform. Social/voting features drive quality. Hybrid of prompt marketplace and character chat platform.

### 3. SnackPrompt

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | 4.0/5 rating; #4 Product of the Day on Product Hunt (July 2025) |
| **Barrier to publish** | LOW - Free registration; upload and share |
| **Direct use by end users** | MEDIUM-HIGH - Browse prompts, use with connected AI services |
| **Monetization** | Buy/sell custom prompts; marketplace features |
| **Traffic/User base** | Growing community |

**Verdict**: Strong community-driven alternative to PromptBase. Focus on quality curation over volume.

### 4. ChatX

| Dimension | Assessment |
|-----------|-----------|
| **Scale** | Free marketplace for ChatGPT, DALL-E, Stable Diffusion, Midjourney prompts |
| **Barrier to publish** | LOW |
| **Direct use by end users** | MEDIUM - Browse and copy prompts |
| **Monetization** | Free marketplace |
| **Chinese presence** | Supports Chinese-language prompts |

**Verdict**: Good free alternative for prompt discovery. Limited monetization for creators.

### 5. Other Notable Prompt Platforms

- **PromptHero**: Dominant for image generation prompts (Midjourney, Stable Diffusion)
- **AiShort (aishort)**: Chinese AI prompt library with community sharing
- **Prompt X3**: Chinese/English prompt tool for ChatGPT, Midjourney, Stable Diffusion

## Part 5: Strategic Recommendations

### For maximum consumer reach (Chinese market):
1. **扣子/Coze** - Publish skills/bots here first. Largest Chinese AI distribution network.
2. **阿里百炼 + 百宝箱** - Best monetization via Alipay tipping. Consumer-friendly.
3. **智谱清言/AutoClaw** - Growing skill ecosystem; GLM-5 model is strong.

### For maximum consumer reach (Global market):
1. **OpenAI GPT Store** - Unmatched user base (700M weekly).
2. **FlowGPT** - Best "try immediately" prompt experience.
3. **SnackPrompt** - Growing community with quality curation.

### For developer/technical distribution:
1. **Claude Code Skills** - Lowest friction skill definition (markdown). Growing fast.
2. **n8n Templates** - 9,000+ templates; strong automation community.
3. **Dify Creator Center** - Open-source workflow sharing.

### For prompt selling/monetization:
1. **PromptBase** - Established marketplace, 80% creator revenue share.
2. **FlowGPT** - Largest community, freemium model.
3. **阿里百炼 AI打赏** - Innovative tipping model via Alipay.

### The meta-trend:
The industry is moving from **"prompt marketplaces"** (selling text strings) to **"skill/agent marketplaces"** (packaging complete workflows). Coze's Skill Store and the GPT Store represent the future direction. Platforms that combine low creation barrier with instant end-user consumption will dominate.

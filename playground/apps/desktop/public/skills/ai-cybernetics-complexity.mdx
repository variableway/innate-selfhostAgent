## 第一部分：控制论 — 反馈创造智能

### 什么是控制论？

控制论（Cybernetics）研究的是**系统如何通过反馈来自我调节**。

最简单的例子：**空调温控**

```
目标温度: 25°C
    ↑
    │  比较
    ↓
实际温度: 28°C → 开冷气 → 温度下降 → 25°C → 关冷气
                                    ↑ 温度上升
                                    └──── 再次循环
```

这是一个**负反馈循环**：偏离目标 → 纠正 → 回到目标。

### AI 中的反馈

| 反馈类型 | 说明 | AI 应用 |
|---------|------|---------|
| 负反馈 | 纠正偏差，回到目标 | 训练中的损失函数 |
| 正反馈 | 放大偏差 | 生成式 AI 的"幻觉"恶化 |
| 延迟反馈 | 效果延迟出现 | 强化学习的长期奖励 |

### 强化学习 — 控制论的AI实现

强化学习（RL）是控制论在 AI 中的直接应用：

```
智能体 (Agent)
  │
  ├── 观察环境状态 → State
  ├── 选择行动 → Action
  ├── 获得奖励/惩罚 → Reward
  └── 更新策略 → Policy
        │
        └── 循环
```

**实际例子**：
- AlphaGo：状态=棋盘，行动=落子，奖励=赢棋
- 自动驾驶：状态=路况，行动=方向盘/刹车，奖励=安全到达

## 第二部分：复杂系统 — 涌现与不可预测性

### 什么是复杂系统？

复杂系统的特点是：**整体行为不能从部分行为简单推导**。

**例子**：
- 单个蚂蚁很简单，但蚂蚁群体能建造复杂的巢穴
- 单个神经元很简单，但大脑能产生意识
- 单个 AI 模型层很简单，但深度网络能产生惊人的能力

这就是**涌现（Emergence）**——简单规则产生复杂行为。

### AI 中的涌现能力

大语言模型展现出的涌现能力让研究者惊讶：

| 模型规模 | 涌现的能力 |
|---------|----------|
| 小模型（小于1B 参数） | 基本语言生成 |
| 中模型（1-10B） | 翻译、问答 |
| 大模型（10-100B） | 逻辑推理、数学、编程 |
| 超大模型（100B+） | 规划、创造性思维 |

**关键发现**：某些能力不是逐渐出现的，而是在特定规模上突然"涌现"。这就像水的沸点——99°C 是水，100°C 突然变成蒸汽。

### 规模法则（Scaling Laws）

AI 性能和资源投入之间存在可预测的关系：

```
性能 ∝ (参数量)^α × (数据量)^β × (计算量)^γ
```

**实际意义**：
- 更大的模型几乎总是更强
- 更多的数据几乎总是更好
- 但投入的边际收益在递减

## 第三部分：信号处理 — AI 的"感官"

### 从信号到信息

AI 处理的一切数据本质上都是**信号**：

| 数据类型 | 原始信号 | AI 处理方式 |
|---------|---------|-----------|
| 文字 | 字符序列 | 嵌入向量 + Transformer |
| 图片 | 像素矩阵 | 卷积 + 池化 |
| 音频 | 声波 | 频谱分析 + 声学模型 |
| 视频 | 图片序列 | 时序模型 + 空间特征 |

### 傅里叶变换 — 拆解信号

傅里叶变换把复杂的信号拆解成简单的正弦波的叠加：

```
复杂信号 = sin(低频) + sin(中频) + sin(高频)
```

这就像把一首歌拆成不同乐器的声音。AI 用类似的原理来理解复杂数据。

## 第四部分：系统工程视角

### 把 AI 看作一个系统

从系统工程角度看，AI 系统包含多个层次：

```
┌────────────────────────────┐
│  应用层：用户交互            │
├────────────────────────────┤
│  模型层：推理和生成          │
├────────────────────────────┤
│  训练层：学习和优化          │
├────────────────────────────┤
│  数据层：收集、清洗、存储     │
├────────────────────────────┤
│  基础设施层：GPU、网络、存储  │
└────────────────────────────┘
```

### 系统性问题的特征

| 特征 | 说明 | AI 的表现 |
|------|------|---------|
| 非线性 | 小输入变化 → 大输出变化 | "蝴蝶效应"式的幻觉 |
| 反馈循环 | 输出影响输入 | AI 对话的上下文依赖 |
| 涌现 | 整体 ≠ 部分之和 | 大模型的意外能力 |
| 路径依赖 | 历史影响未来 | 训练数据的偏差传递 |

# 简单的反馈控制系统：温度调节器
class Thermostat:
    def __init__(self, target=25):
        self.target = target
        self.current = 15  # 初始温度
        self.history = []

    def step(self):
        error = self.target - self.current
        adjustment = error * 0.3  # 比例控制
        noise = (hash(str(self.current)) % 10 - 5) * 0.1  # 环境噪声
        self.current += adjustment + noise
        self.history.append(round(self.current, 2))

thermo = Thermostat(target=25)
print('温度调节器反馈过程：')
for i in range(15):
    thermo.step()
    status = '✓ 到达目标' if abs(thermo.current - 25) < 0.5 else '...'
    print(f'  第{i+1}步: {thermo.current:.1f}°C {status}')

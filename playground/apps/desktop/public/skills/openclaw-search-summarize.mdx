# AI 助手基本使用 — 搜索和总结

> 信息过载是现代人的常态。每天有无数文章、报告、新闻要处理。本教程教你让 AI 助手帮你搜索信息、总结长文，把信息处理时间从 1 小时缩短到 5 分钟。

---

## 前置要求

- 已完成 [OpenClaw 5分钟体验](#) 教程
- OpenClaw 已安装并正常运行

---

## 步骤 1：安装搜索技能

OpenClaw 使用技能系统来扩展能力。搜索功能需要安装 `tavily-search` 技能：

<RunnableCodeBlock code="claw skill install tavily-search" language="sh" />

安装后需要配置搜索 API Key（免费额度足够日常使用）：

1. 访问 [tavily.com](https://tavily.com) 注册账号
2. 在控制台获取 API Key
3. 运行配置命令：

<PlatformTabs
  unix={<RunnableCodeBlock code="claw config set TAVILY_API_KEY=你的API密钥" language="bash" />}
  windows={<RunnableCodeBlock code="claw config set TAVILY_API_KEY=你的API密钥" language="powershell" />}
/>

验证技能是否加载：

<RunnableCodeBlock code="claw skills list | grep search" language="sh" />

---

## 步骤 2：使用搜索功能

启动 OpenClaw 并尝试搜索：

<RunnableCodeBlock code="claw chat" language="sh" />

在对话中使用搜索：

```
帮我搜索 2026 年最值得学习的 AI 编程工具
```

OpenClaw 会自动识别这是一个搜索请求，调用搜索技能，返回结果。

### 搜索技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 直接提问 | AI 会自动搜索 | "Python 和 Go 哪个更适合 AI 开发？" |
| 指定范围 | 限定搜索领域 | "在 arxiv 上搜索最新的 Transformer 论文" |
| 对比搜索 | 同时搜索多个话题 | "对比 Claude Code 和 Cursor 的优缺点" |
| 追问 | 基于结果深入 | "告诉我更多关于第一点的细节" |

## 步骤 3：使用总结功能

总结功能不需要额外安装，OpenClaw 自带文本处理能力。

### 总结网页内容

```
请总结这个页面的核心内容：https://example.com/article
```

### 总结本地文件

```
请总结 ~/Documents/report.pdf 的要点
```

### 总结粘贴的文本

直接把长文章粘贴到对话中：

```
请帮我总结以下文章的 3 个核心要点：

[粘贴文章内容...]
```

### 总结输出格式

你可以指定总结的格式：

```
请用以下格式总结这篇文章：
- 一句话概述
- 3个关键要点
- 2个值得注意的细节
- 1个行动建议
```

---

## 步骤 4：组合使用 — 搜索 + 总结

最强大的用法是把搜索和总结组合起来：

```
帮我搜索最近关于 RISC-V 的新闻，然后总结最重要的 3 条
```

```
搜索 React 19 的新特性，用表格对比 React 18 的主要变化
```

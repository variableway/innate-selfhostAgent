# 认识 AI 助手 — OpenClaw 5分钟体验

> 你可能已经用过 ChatGPT 或其他 AI 聊天工具，但你知道吗？AI 还可以像一个全能助手一样，帮你搜索信息、管理邮件、安排日程、甚至写代码。本教程带你 5 分钟体验这一切。

---

## AI 助手是什么？

想象你有一个不知疲倦的助手，它：

- **随时在线** — 不需要休息，24/7 待命
- **多才多艺** — 搜索信息、写邮件、管理日程、写代码样样行
- **理解你的话** — 用自然语言沟通，不需要学命令行

**OpenClaw** 就是这样一个 AI 助手系统。它运行在你的电脑上，连接各种 AI 模型，通过"技能"来扩展能力。

### OpenClaw 能做什么？

| 能力 | 说明 | 示例 |
|------|------|------|
| 搜索 | 联网搜索最新信息 | "帮我查一下今天的AI新闻" |
| 写邮件 | 读取、起草、发送邮件 | "帮我回复这封邮件" |
| 管理日程 | 日历提醒和安排 | "明天有什么安排？" |
| 写代码 | 理解和生成代码 | "帮我写一个Python脚本" |
| 浏览器 | 控制浏览器完成操作 | "帮我在淘宝搜索..." |
| 文档总结 | 长文章一键总结 | "总结这篇论文" |

## 步骤 1：一键安装 AutoClaw

AutoClaw 是 OpenClaw 的一键安装工具，5 分钟完成所有配置。

<PlatformTabs
  unix={<RunnableCodeBlock code="curl -fsSL https://autoclaw.com/install.sh | bash" language="bash" />}
  windows={<RunnableCodeBlock code="irm https://autoclaw.com/install.ps1 | iex" language="powershell" />}
/>

安装过程会自动完成：
- 检测你的操作系统
- 安装 Node.js（如果没有）
- 安装 OpenClaw 核心组件
- 配置默认 AI 模型
- 安装 50+ 预装技能

---

## 步骤 2：第一次对话

安装完成后，在终端中启动 OpenClaw：

<RunnableCodeBlock code="claw chat" language="sh" />

你会看到类似这样的界面：

```
🤖 OpenClaw v2026.3
已连接模型: glm-5.1
已加载技能: 52 个

> 你好！我是你的 AI 助手，有什么可以帮你的？
```

试试输入以下内容：

```
你好，请介绍一下你自己
```

---

## 步骤 3：体验一个技能

让我们试试搜索技能。在对话中输入：

```
帮我搜索一下 "2026年最热门的AI工具"
```

OpenClaw 会自动调用搜索技能，返回结果。你也可以试试：

```
现在几点了？今天天气怎么样？
```

---

## 步骤 4：验证安装

运行以下命令确认一切正常：

<RunnableCodeBlock code="claw --version && claw skills list" language="sh" />

如果看到版本号和技能列表，说明安装成功！

## 第一部分：微积分 — AI 如何"学习"

### 什么是导数？

想象你在爬山，站在山坡上，想知道哪个方向下坡最快。

**导数就是"坡度"** — 它告诉你当前方向是上坡还是下坡，坡度有多陡。

- 导数大于 0 → 上坡（函数在增大）
- 导数小于 0 → 下坡（函数在减小）
- 导数等于 0 → 平坦（可能是山顶或山谷）

### 梯度下降：下山的故事

AI 学习的过程，本质上就是**蒙着眼睛下山**：

1. 你站在山坡上某个随机位置（模型初始参数）
2. 用脚感受地面的坡度（计算导数/梯度）
3. 朝着下坡最陡的方向走一步（更新参数）
4. 重复，直到走到谷底（找到最优解）

```
起点 ╲
      ╲  ← 每一步都朝着"下坡"方向走
       ╲
        ╲
         ● ← 到达谷底（最优解）
```

这就是 AI 学习的核心算法——**梯度下降（Gradient Descent）**。

### 学习率 — 每步走多大

下山时，步子太大会跳过谷底，步子太小走得太慢。这个"步子大小"就是**学习率**。

| 学习率 | 效果 |
|--------|------|
| 太大 | 跳过最优解，来回震荡 |
| 太小 | 收敛太慢，训练时间过长 |
| 合适 | 稳定快速地找到最优解 |

### 用代码体验梯度下降

<RunnableCodeBlock code={`python3 -c "
# 简单的梯度下降演示
# 找到函数 f(x) = x^2 的最小值（答案是 x=0）
x = 5.0  # 起点
lr = 0.1  # 学习率
print('梯度下降过程：')
for i in range(10):
    grad = 2 * x  # f(x)=x^2 的导数是 2x
    x = x - lr * grad
    print(f'  第{i+1}步: x = {x:.4f}')
print(f'最终结果: x = {x:.4f} (理论最优解: x = 0)')
"`} language="python" />

## 第二部分：信息论 — AI 如何"理解"信息

### 什么是信息量？

想象两个场景：

- **场景 A**："明天太阳会从东边升起" — 这不是新闻，信息量很小
- **场景 B**："明天有陨石撞击地球" — 这是大新闻，信息量很大

信息量的大小取决于**事情有多出乎意料**：
- 越不可能发生的事情，包含的信息量越大
- 一定会发生的事情，信息量为零

### 熵 — 不确定度的度量

**熵**衡量的是一个系统的"混乱程度"或"不确定度"。

**例子**：
- 抛一枚均匀硬币：熵最高（完全不确定）
- 抛一枚两面都是正面的硬币：熵为零（完全确定）
- 抛一枚偏心的硬币：熵在两者之间

```
均匀硬币  ████████████  熵 = 1.0（最不确定）
偏心硬币  ██████        熵 = 0.7
双面相同  ▏             熵 = 0.0（完全确定）
```

### AI 为什么关心熵？

1. **决策**：AI 用熵来判断该关注什么。高熵=不确定=需要更多信息
2. **压缩**：找到规律就能压缩数据，熵告诉我们可以压缩多少
3. **学习**：AI 的目标通常是减少系统的不确定性（降低熵）

### 用代码体验信息熵

<RunnableCodeBlock code={`python3 -c "
import math
# 计算抛硬币的熵
def entropy(p):
    if p == 0 or p == 1:
        return 0
    return -p * math.log2(p) - (1-p) * math.log2(1-p)

print('不同概率下的信息熵：')
for p in [0.5, 0.7, 0.9, 0.99, 1.0]:
    print(f'  正面概率={p:.0%} → 熵={entropy(p):.3f} bit')
print()
print('结论：越不确定，熵越高！')
"`} language="python" />

## 四个数学工具如何协作

现在我们把概率、线性代数、微积分、信息论串起来，看看 AI 的完整工作流：

```
┌─────────────────────────────────────────────┐
│  输入：一张猫的图片                            │
├─────────────────────────────────────────────┤
│  1. 线性代数：把图片变成向量 [0.1, 0.8, ...]   │
│  2. 矩阵运算：通过神经网络层处理                  │
│  3. 概率论：输出 [猫:0.92, 狗:0.05, ...]       │
│  4. 信息论：计算预测的熵，判断是否自信            │
│  5. 微积分：如果预测错了，用梯度下降调整参数       │
├─────────────────────────────────────────────┤
│  输出：这是猫，92% 把握                          │
└─────────────────────────────────────────────┘
```

### 数学工具速查表

| 工具 | 一句话 | AI 中的角色 |
|------|--------|-----------|
| 概率论 | 衡量可能性 | 判断、预测、不确定性 |
| 线性代数 | 用数字表示世界 | 数据表示、批量计算 |
| 微积分 | 找到最优方向 | 学习（梯度下降） |
| 信息论 | 衡量信息量 | 评估、压缩、决策 |

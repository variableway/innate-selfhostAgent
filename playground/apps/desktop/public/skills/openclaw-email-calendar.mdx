> 你每天花多少时间处理邮件？30 分钟？1 小时？如果 AI 能帮你读邮件、起草回复、管理日程，你就能把时间用在更重要的事情上。

## 步骤 1：连接邮箱

### 安装邮件技能

<RunnableCodeBlock code="claw skill install gog" language="sh" />

### 配置邮箱连接

OpenClaw 支持通过 IMAP 协议连接大多数邮箱。

**Gmail 用户**：

1. 在 Google 账号设置中开启"应用专用密码"
2. 运行配置命令：

<PlatformTabs
  unix={<RunnableCodeBlock code="claw config set EMAIL_IMAP_HOST=imap.gmail.com && claw config set EMAIL_ADDRESS=你的邮箱@gmail.com && claw config set EMAIL_PASSWORD=你的应用专用密码" language="bash" />}
  windows={<RunnableCodeBlock code="claw config set EMAIL_IMAP_HOST=imap.gmail.com; claw config set EMAIL_ADDRESS=你的邮箱@gmail.com; claw config set EMAIL_PASSWORD=你的应用专用密码" language="powershell" />}
/>

**163 邮箱用户**：

1. 在 163 邮箱设置中开启 IMAP 并获取授权码
2. 运行配置命令：

<PlatformTabs
  unix={<RunnableCodeBlock code="claw config set EMAIL_IMAP_HOST=imap.163.com && claw config set EMAIL_ADDRESS=你的邮箱@163.com && claw config set EMAIL_PASSWORD=你的授权码" language="bash" />}
  windows={<RunnableCodeBlock code="claw config set EMAIL_IMAP_HOST=imap.163.com; claw config set EMAIL_ADDRESS=你的邮箱@163.com; claw config set EMAIL_PASSWORD=你的授权码" language="powershell" />}
/>

### 验证邮箱连接

<RunnableCodeBlock code="claw chat -m '检查我的邮箱连接是否正常，读取最近 3 封邮件的标题'" language="sh" />

如果看到邮件标题，说明连接成功。

## 常见问题

### Q: 邮箱连接失败？

检查：
1. IMAP 是否已开启（在邮箱设置中）
2. 是否使用了应用专用密码（不是登录密码）
3. 网络是否能访问 IMAP 服务器

### Q: 163 邮箱需要授权码？

是的，163 邮箱不能使用登录密码。在 163 邮箱的"设置 → POP3/SMTP/IMAP"中开启并获取授权码。

### Q: 邮件回复会自动发送吗？

不会。AI 会先展示草稿，你确认后才会发送。你可以说"发送"或"修改一下再发"。
